📖✨ Capibooks — Clube do Livro

Esta página foi criada para apresentar o Capibooks, um clube do livro idealizado para conectar leitores apaixonados por boas histórias e conversas significativas 💬📚

O Capibooks nasceu com o propósito de transformar cada leitura em uma experiência coletiva, incentivando a troca de ideias, diferentes perspectivas e o aprofundamento em novas narrativas. Mais do que ler, o clube é sobre conexão, aprendizado e a construção de uma comunidade unida pelo amor aos livros 🤎

💻🚀 Desenvolvimento Front-End

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, com foco em organização, responsividade e boa experiência do usuário.

HTML5 🧱: Estruturação semântica da página, priorizando acessibilidade e hierarquia clara das informações.

CSS3 🎨: Estilização moderna com uso de Flexbox/Grid para layout responsivo, garantindo adaptação a diferentes dispositivos 📱💻.

JavaScript ⚡: Implementação de interações dinâmicas e comportamentos que tornam a navegação mais fluida e intuitiva.

Durante o desenvolvimento, foram aplicados princípios de UI/UX para proporcionar uma navegação agradável, clara e objetiva, alinhando estética e funcionalidade.
